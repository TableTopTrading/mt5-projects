# SuperSlopeDashboard Indicator User Guide
## Overview 
The SuperSlopeDashboard is a custom technical analysis tool for MetaTrader 5 that provides a real-time visual dashboard displaying multiple currency pairs categorized by their trend strength. It uses the same SuperSlope calculation methodology as the standalone SuperSlope indicator but extends it to monitor multiple symbols simultaneously. 

**Key Features:** 
- Real-time monitoring of multiple symbols 
- Color-coded strength categorization (5 categories) 
- MVC (Model-View-Controller) architecture for reliability 
- Customizable thresholds and parameters 
- On-chart visual dashboard display

## Installation 
### Required Files 
The SuperSlopeDashboard indicator requires **four files** to function properly: 
1. **SuperSlopeDashboard.mq5** - Main indicator file 
2. **CDashboardController_v2.mqh** - Controller class (MVC architecture) 
3. **CDataManager.mqh** - Model class (data calculations) 
4. **CRenderer.mqh** - View class (dashboard rendering) 
### File Placement 
Copy the files to the following locations in your MetaTrader 5 directory: 
```
📁 [MetaTrader 5 Data Folder] └── 📁 MQL5 ├── 📁 Indicators │ └── 📁 MyProjects │ └── 📁 SuperSlopeDashboard │ └── 📄 SuperSlopeDashboard.mq5 └── 📁 Include └── 📁 MyProjects └── 📁 SuperSlopeDashboard ├── 📄 CDashboardController_v2.mqh ├── 📄 CDataManager.mqh └── 📄 CRenderer.mqh 
```
 **To find your MetaTrader 5 Data Folder:** 
 - Open MetaTrader 5 - Go to File → Open Data Folder - Navigate to MQL5 folder 
### Installation 
Steps 
1. Create the folder structure as shown above if it doesn't exist 
2. Copy `SuperSlopeDashboard.mq5` to the `MQL5/Indicators/MyProjects/SuperSlopeDashboard/` folder 
3. Copy the three include files to the `MQL5/Include/MyProjects/SuperSlopeDashboard/` folder 
4. Restart MetaTrader 
5. 5 or press **Ctrl+F5** to refresh 
6. The indicator will appear in Navigator under "Custom Indicators" → "MyProjects" → "SuperSlopeDashboard"

## What the Dashboard Displays 
### Visual Layout 
The SuperSlopeDashboard displays a five-column dashboard on your chart: 
``` ┌─────────────────┬─────────────────┬─────────────────┬─────────────────┬─────────────────┐ │ STRONG BULL │ WEAK BULL │ NEUTRAL │ WEAK BEAR │ STRONG BEAR │ ├─────────────────┼─────────────────┼─────────────────┼─────────────────┼─────────────────┤ │ EURUSD 2.15 │ GBPUSD 0.87 │ USDCHF 0.12 │ USDCAD -0.63 │ USDJPY -2.34 │ │ AUDUSD 1.92 │ NZDUSD 0.75 │ │ │ │ └─────────────────┴─────────────────┴─────────────────┴─────────────────┴─────────────────┘ ``` 
### Calculation Method 
The dashboard uses the same **SuperSlope calculation** as the standalone indicator: 
1. **Moving Average Calculation:** Uses a Linear Weighted Moving Average (LWMA) of close prices
2. **Trend Rate Calculation:** Measures rate of change between current and previous MA values 
3. **ATR Normalization:** Divides trend rate by Average True Range (ATR) to normalize for volatility 
4. **Smoothing:** Applies weighted smoothing formula for stability **Mathematical Formula:** 

```
Slope = (CurrentMA - SmoothedPreviousMA) / (ATR / 10) Where: - CurrentMA = LWMA(Close, SlopeMAPeriod)[current_bar] - SmoothedPreviousMA = (LWMA[previous_bar] × 231 + ClosePrice × 20) / 251 - ATR = Average True Range over SlopeATRPeriod 
``` 
### Strength Categorization 
Symbols are categorized into five strength categories based on configurable thresholds: 
| Category | Threshold Range | Default Color | Interpretation | 
|----------|-----------------|---------------|----------------| 
| **Strong Bull** | ≥ Strong Threshold | Lime Green | Very strong upward momentum | 
| **Weak Bull** | ≥ Weak Threshold but < Strong Threshold | Green | Moderate upward momentum | 
| **Neutral** | Between -Weak Threshold and +Weak Threshold | Yellow | Sideways/consolidation | 
| **Weak Bear** | ≤ -Weak Threshold but > -Strong Threshold | Orange | Moderate downward momentum | 
| **Strong Bear** | ≤ -Strong Threshold | Red | Very strong downward momentum |

Parameters | Parameter | Type | Default | Description | 
|-----------|------|---------|-------------| 
| **Slope MA period** | Integer | 7 | Period for Linear Weighted Moving Average calculation |

| **Max bars to calculate** | Integer | 500 | Maximum historical bars to process (0 = all available data) | 
### Dashboard

|-----------|------|---------|-------------|

| **Weak Threshold** | Double | 0.5 | Threshold for Weak Bull/Bear categorization |
